## Usage
* To create environment
```
python3 -m venv env
```
* To launch environment
```
source ./env/bin/Activate
```
* To install necessary packages
```
pip3 install -r requirements.txt
```
* To run the game
```
python3 game.py [args]
```
* Example
```
python3 game.py rock paper scissors
```
